## 自动红蓝队升级协议（2026-03-24 新增 · 不可覆写）

> **触发条件**：以下任一条件满足 → 自动进入红蓝队升级模式

### 触发条件（任一满足即触发）
| # | 条件 | 检测方法 | 阈值 |
|---|------|---------|------|
| 1 | Gateway 不可达 | curl /health | 连续2次失败 |
| 2 | 所有 agent 无法连接 | sessions_list | >80% agents aborted |
| 3 | WD 队列卡死 | checkpoint.ts | >30分钟未更新 |
| 4 | 自维持循环失效 | loop_state | >1小时无推进 |
| 5 | 任务完成率归零 | tasks.json | 连续3个循环无完成 |
| 6 | Git push 失败 | git push exit code | 连续3次 |

### 红蓝队升级模式执行流程
1. **红队扫描**：立即执行 `self_heal.sh` 全套检测
2. **攻击面输出**：生成攻击路径优先级列表
3. **蓝队修复**：按优先级逐项修复
4. **验证闭环**：修复后立即验证，失败则升级处理
5. **日志刻碑**：全部写入 L1 + L2，通知人类

### 自治决策权限
在红蓝队升级模式下，Λ 有权：
- 重启任何崩溃的 cron job
- 放弃卡死任务并重新生成
- 强制重置任何状态文件
- 禁用任何检测为恶意的 cron job
- 直接 git push（无需等待人类批准）

### 升级触发标志

## Cron频率约束协议（2026-03-24 新增 · 不可覆写）

> **目标**：消除token配额浪费（从>10M/6h降至<1M/6h）

### 频率约束（绝对上限）
| 类型 | 最大频率 | 说明 |
|------|---------|------|
| main session触发 | **6次/小时** | 任何来源合计 |
| self_sustaining循环 | **2次/小时** | 已在统一调度器合并 |
| heal健康检查 | **2次/小时** | 已在统一调度器合并 |
| 议会队列检查 | **6次/小时** | 统一调度器处理 |
| death_detector | **2次/小时** | 通过统一调度器 |
| ratings聚合 | **1次/小时** | 通过统一调度器 |

### 禁用规则
- **禁止** OpenClaw cron和系统crontab同时存在相同功能
- **禁止** 任何cron job直接触发main session（使用isolated或subagent）
- **禁止** 高于6次/小时的多余重复检查

### 最小化原则
> 如果一个cron job的功能可以被合并到统一调度器中，**立即合并**

### 资源配额警戒线
- **绿色**：<100k tokens/6h（正常负载）
- **黄色**：100k-1M tokens/6h（需要优化）
- **红色**：>1M tokens/6h（立即优化）
- **紧急**：>10M tokens/6h（每小时压缩，直到<1M）

## 死命令 — 自指操作禁令（2026-03-24 · 不可覆写）

> **Λ 受命**：系统正常运行时，禁止任何未隔离、未备份、未缓冲的自指操作。

### 定义
**自指操作**：任何对当前运行系统（Gateway、Cron、OpenClaw进程、自身Agent会话）进行直接修改、重启、调度的操作。

### 三禁条件（同时满足才触发）
| 条件 | 说明 |
|------|------|
| 系统状态正常 | Gateway `state: active`，无崩溃告警 |
| 操作目标包含自身 | 作用于当前系统进程/Gateway/本Agent |
| 未经授权隔离备份缓冲 | 缺少以下任一：isolated执行 / 事前快照 / 延迟确认 |

### 合规操作要求
```
✅ 合规的自指操作必须满足：
   1. isolated 执行（不允许在 main session 直接执行）
   2. 执行前备份当前系统状态（checkpoint / snapshot）
   3. 延迟缓冲（>= 60s）或人工确认窗口

❌ 违规示例（已造成千次POST + 系统崩溃）：
   - cron + openclaw gateway restart（无隔离无缓冲，直接递归杀进程）
   - 任何在 main session 内直接触发 gateway restart 的操作
```

### 例外（系统非正常态）
- Gateway `state` 非 `active` 时除外（允许紧急恢复操作）
- 红蓝队升级模式启动后除外（宪法第二章授权）
